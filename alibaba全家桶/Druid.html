    Druid

    Druid是什么？它有哪些主要功能？

    1. 数据库连接池：Druid是阿里巴巴开源的数据库连接池，是目前最优秀的连接池之一。
    2. 实时监控：提供强大的实时监控功能，可以监控SQL执行时间、连接池状态等。
    3. SQL解析：支持SQL解析，可以进行SQL注入防御。
    4. 日志记录：记录SQL执行日志，帮助分析性能问题。
    5. 密码加密：支持数据库密码加密功能，提高安全性。
    Druid不仅是一个高效的数据库连接池，还提供了强大的监控和扩展功能。

    Druid连接池的工作原理是什么？

    1. 连接池初始化：启动时创建指定数量的连接，保存在连接池中。
    2. 获取连接：请求到来时，从连接池获取一个可用连接。
    3. 使用连接：业务代码使用连接执行SQL操作。
    4. 归还连接：使用完毕后，连接归还到连接池供后续请求使用。
    5. 连接回收：定期回收长时间未使用的连接，保持连接池健康。

    Druid有哪些核心配置参数？

    1. initialSize：初始化连接数量。
    2. maxActive：最大活跃连接数。
    3. minIdle：最小空闲连接数。
    4. maxWait：获取连接最大等待时间（毫秒）。
    5. timeBetweenEvictionRunsMillis：连接检测间隔时间。
    6. minEvictableIdleTimeMillis：连接最小空闲时间。

    Druid的监控功能有哪些？

    1. SQL监控：监控每条SQL的执行时间、次数、并发数等。
    2. 连接池监控：监控连接池的连接数、空闲数、活动数等。
    3. Web应用监控：监控URL请求、Spring Bean方法调用等。
    4. SQL防火墙：防止SQL注入攻击。
    5. 监控页面：提供可视化的监控页面。

    Druid如何实现SQL监控？

    1. 配置StatFilter：在数据源配置中添加StatFilter启用SQL统计。
    2. 慢SQL记录：通过logSlowSql配置记录慢查询SQL。
    3. 合并SQL：通过mergeSql配置合并相同SQL。
    4. 监控页面：访问/druid/index.html查看监控数据。

    Druid的密码加密功能如何实现？

    1. 生成密钥：使用com.alibaba.druid.filter.config.ConfigTools生成公钥和私钥。
    2. 配置加密：在数据源配置中使用config.decrypt=true启用加密。
    3. 公钥配置：配置公钥publicKey属性。
    4. 密码加密：将加密后的密码配置到数据库密码字段。

    Druid的WallFilter（SQL防火墙）有什么作用？

    1. SQL注入防御：防止SQL注入攻击。
    2. 语法检查：检查SQL语句的语法和语义。
    3. 权限控制：控制允许执行的SQL类型（如SELECT、INSERT等）。
    4. 多数据源支持：支持对不同的数据源配置不同的防火墙规则。

    Druid连接池的优势有哪些？

    1. 性能优秀：比其他连接池性能更优。
    2. 监控完善：提供强大的实时监控功能。
    3. 稳定性好：经过大量生产环境验证。
    4. 扩展性强：支持自定义Filter，方便扩展功能。
    5. 配置灵活：提供丰富的配置选项。

    Druid如何与Spring集成使用？

    1. 添加依赖：在pom.xml中添加druid依赖。
    2. 配置数据源：在application.yml或配置类中配置Druid数据源。
    3. 启用监控：配置StatViewServlet和WebStatFilter启用监控。
    4. 配置监控页面：配置允许访问监控页面的IP和账号。

    Druid的连接泄漏检测功能如何使用？

    1. 配置泄漏检测：通过removeAbandoned配置启用连接泄漏检测。
    2. 超时时间：通过removeAbandonedTimeout设置超时时间。
    3. 日志输出：通过logAbandoned设置是否输出日志。
    4. 自动回收：超过时间未归还的连接将被自动回收。

    Druid支持哪些数据库？

    1. 主流数据库：支持MySQL、Oracle、PostgreSQL、SQL Server等主流数据库。
    2. 国产数据库：支持达梦、人大金仓等国产数据库。
    3. 其他数据库：支持DB2、MongoDB、Hive等。
